# landing-page

صفحة هبوط بسيطة مرتبطة بنموذج Google Form لتجميع بيانات (اسم، بريد إلكتروني، وصف المشكلة).

## 📌 المميزات
- تصميم خفيف وسريع.
- إرسال البيانات مباشرة إلى Google Form عبر رابط `formResponse`.
- الردود محفوظة تلقائيًا في Google Sheets.

## 🚀 كيفية الاستخدام
1. افتح الرابط: [https://kimo-deep.github.io/landing-page/](https://kimo-deep.github.io/landing-page/)
2. املأ الحقول المطلوبة.
3. اضغط زر "إرسال" ليتم حفظ البيانات في النموذج.
